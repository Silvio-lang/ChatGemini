Treinamento para deficiente: Toque na tela da direita para acelerar à direita. Na da esquerda para acelerar à esquerda. O som para quando estiver na faixa central. O som intermitente é porque está se deslocando para direita ou esquerda. A frequência mais alta é para a direita. Som contínuo é porque já saiu muito fora da região e permanece até chegar ao centro. Este treinamento deve auxiliar para a atividade prática.
